Game Rules

2048 is a single-player sliding block puzzle game designed by Italian web developer Gabriele Cirulli. The game's objective is to slide numbered tiles on a grid to combine them and create a tile with the number 2048.

When you slide, all tiles on the grid will combine to create a new tile if they share the same number and are touching. If two identical tiles are combined, they will create a new tile with the number twice as large as the original ones.

Every turn, a new tile with a value of either 2 or 4 will randomly appear on the grid. If all positions on the grid are filled and no more moves can be made, the game is over.
